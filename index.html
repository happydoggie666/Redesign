Part 1: Identifying Usability Problem

1.1 Picking a Web Page

Although being ugly is one of its special features and making the website ‘attractive’, the overly crowded text, the overly colorful layout and the overused animation is very overwhelming for both new users and old users.

1.2 Finding Problems

1.2.1 Mental v.s. Conceptual model

Mental Model:

When entering lingsCars trying to rent a car, people would mostly think of:

  1. Look at the menu
  2. Choose the car brand they like or directly click on the specific car they are interested in to see the detail
  3. In the detail session, they would like to see the price and the related contract
  4. Check the review & the information contact of LingsCars
Conceptual model:

The conceptual model that this website provides is:

  1. Provide a large banner which leads to the car list
  2. Show the menu
  3. Provide the car brand
  4. Show the contact information

1.2.2 Usability & Efficiency

  1. Low image-font color contrast
  2. Hard for both new users and old users to spot important information and quickly perform the operation they want
  3. Many photos have the same alt text, which cannot provide much information if the image is not showing
  4. The large empty white region in the bottom

1.2.3 Memorability

Although the website provides many affordances with the usage of icons, it is still hard to recall how to do a single operation. Say, the user wants to rent a car, after they click the “order now” icon, it is hard to tell what should be done next.

1.2.4 Learnability

Although the website provides many affordances with the usage of icons, it is still hard to learn for new users. The overly colorful page and overly crowded icons are just so distracting.

1.3 Accessibility

WebAIM WAVE Report

Accessibility findings:

The website’s main problem is it has many (46) contrast errors and the heading hierarchy/structure is very unclear. There are many redundant and small texts which are not easily recognizable. WAVE also finds that a nearby image contained in the website has the same alternative text, which cannot provide hints for the users of what the image is about if the image is not showing.

My personal opinion:

I agree with the WAVE’s opinion that the website has many contrast errors, the overly colorful page makes it hard for users to find something they are looking for efficiently. Although there are some icons which support affordance, it is still not that distinguishable. The page hierarchy is not clear enough. Many description texts are redundant. Also, the website seems to use too many LING’s portraits and memes.
